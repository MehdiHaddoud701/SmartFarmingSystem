# Resource Management Optimization in Smart Farming

## Overview

This project presents an AI-based system for optimizing resource management in smart farming. The goal is to efficiently allocate water, fertilizer, and irrigation while maximizing crop yield and maintaining sustainability.

The system applies Greedy Search, A* Search, Genetic Algorithms, and Constraint Satisfaction Problems (CSP) to determine optimal resource usage based on environmental and soil conditions.

---

## Objectives

Minimize water, fertilizer, and irrigation usage  
Maximize crop yield  
Ensure sustainable farming practices  

---

## Dataset

The project uses the Smart Farming 2024 (SF24) dataset.

### Main features include:

Soil properties: Nitrogen, Phosphorus, Potassium  
Climate data: Temperature, Humidity, Rainfall, Sunlight  
Water usage: Soil moisture, irrigation frequency, efficiency  
Crop data: Crop type, growth stage, pest pressure  
Farming resources: Fertilizer rates, irrigation needs, water sources  

---

## Methods

### Greedy Search

Selects the best immediate resource allocation based on current conditions.

### A* Search

Explores multiple solutions using a cost and heuristic function to find an optimal allocation.

### Genetic Algorithms

Improves solutions over generations using selection, crossover, and mutation.

### Constraint Satisfaction Problem (CSP)

Ensures all constraints are satisfied while optimizing resource allocation.
